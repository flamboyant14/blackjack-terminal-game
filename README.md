# Proyecto 1. Blackjack

Este es mi proyecto final para mi clase de Fundamentos de Programación en el TecMilenio. 
Puse en práctica todos los conceptos y herramientas que nos enseñaron para diseñar mi versión de un sistema que usaría un casino para la compra y venta de fichas, un sistema para apostar, un sistema de la lógica del juego y un sistema para registrar las transacciones. Estos sistemas vienen y trabajan juntos para entregarle al usuario una experiencia simple y divertida donde pueden simular una visita al casino. 

Como correr el programa:
1. Descargar archivo
2. Usar el comando 'python' o 'python3' en la terminal junto con la dirección al archivo en tu computadora.
3. Seguir instrucciones en la terminal para jugar.
4. Al terminar de jugar, salir del programa y revisar el archivo de transacciones para un registro de todos los movimientos.

Tecnologías usadas:
- Python
